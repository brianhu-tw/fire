# 幾歲能退休？

一直上班存錢，到底什麼時候才能退休啊？

## 功能

- 輸入年齡、資產、收入、支出，即時計算最早可退休年齡
- 悲觀 / 基準 / 樂觀三種報酬率情境模擬
- 互動式圖表：拖拉調整退休年齡與收入
- 所有金額以實質購買力（扣除通膨）呈現
- 輸入自動存入瀏覽器，重新整理不會消失

## 預設值

| 欄位 | 預設值 | 來源 |
|------|--------|------|
| 年收入 | NT$ 546,000 | 113 年台灣薪資中位數（主計總處） |
| 年支出 | NT$ 320,000 | 113 年台灣每人消費支出（主計總處家庭收支調查） |
| 年報酬率 | 7% | — |
| 通膨率 | 3% | — |

## 使用方式

純靜態網頁，不需要安裝任何東西。直接用瀏覽器開啟：

```
open index.html
```

或啟動一個本地伺服器：

```
python3 -m http.server
```

## 技術

- 單一 `index.html`，無建置步驟
- [Chart.js 4.x](https://www.chartjs.org/)（CDN 載入）
- CSS custom properties 深色主題
- localStorage 儲存使用者輸入
